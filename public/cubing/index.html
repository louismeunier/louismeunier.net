cubing

I've been speedcubing since 2015, and competing since 2016. In that time, I've found a large number of useful resources, which I've compiled here.

  • Jonatan Kłosko's WCA statistics: Huge variety of WCA statistics created by Jonatan Kłosko
  • 4x4 L2E: Useful last 2 edges algorithms for 4x4
  • Cubing Github: The open source community for cubing
  • CubingChina Statistics: Variety of cubing statistics by CubingChina
  • Nemesizer: Great site comparing competitors' results
  • Sarah's Cubing Site: Sarah Strong's site, with a huge variety of useful algorithms
  • 4x4 POLL: Kris De Asis's Partity OLL algorithms for 4x4
  • Reconstructions: Incredibly useful tool for reconstructing solves
  • Squanmate: Square-1 algorithm trainer and resources
  • PR Streak Spreadsheet: Auto-updating PR streak spreadsheet (made by me!)
